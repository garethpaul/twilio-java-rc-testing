# Getting Started

You'll need to setup environment variables for the items detailed in the Main Class.

```aidl
TWILIO_ACCOUNT_SID
TWILIO_AUTH_TOKEN
TWILIO_PHONE_NUMBER
NGROK_BASE_URL
```

## Compiling and Running

``mvn clean install``

``java -jar target/Testing1234-1.0-jar-with-dependencies.jar
``
